Key Points:
  • User is the base entity with inheritance support (JOINED strategy)
  • Task has lifecycle: ASSIGNED → IN_PROGRESS → COMPLETED
  • ChatMessage supports bidirectional communication with role-based filtering
  • Notification tracks read/unread status for user awareness

  1. Creation: Admin creates a task with title and description
